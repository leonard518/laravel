## Crear rama 
git checkout -b nombre_rama

## Agregar los cambios 
git add .

## Agregar commit y comentario
git commit -m "Mensaje..."

## Subir rama al repositorio
git push origin nombre_rama

## Cambiar a rama master
git checkout master

## Mezclar ramas
git merge nombre_rama

## Subir cambios al repositorio
git push origin
